# HDSL
These are the code files for paper `Low-rank assisted high-dimensional subgroup learning'.
The programs contain three parts:

- simulation_setting.R: Generate simulation data for Scenario 1~5.

- HDSL.R:  Code for estimating the HDSL, HDSL$_V$, and FMM$_{ENET}$ models, as well as for evaluating the results and performing prediction.

- example.R: An example code.

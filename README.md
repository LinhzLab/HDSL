# HDSL
These are the code files for the paper **"Low-rank assisted high-dimensional subgroup learning"**.

The programs contain three parts:

- `simulation_setting.R`: Generate simulation data for **Scenario 1–5** (updated scenarios or new settings).
- `HDSL.R`: Code for estimating HDSL, HDSL$_V$, and FMM$_{ENET}$ models, evaluation, and prediction.
- `example.R`: Example usage script.

# HDSL
These are the code files for the paper **"Low-rank assisted high-dimensional subgroup learning"**.

The programs contain three parts:

- `simulation_setting.R`: Generate simulation data for Scenario 1–5.
- `HDSL.R`: Code for estimating HDSL, HDSL<sub>V</sub> and FMM<sub>ENET</sub> models, as well as for evaluating the results and performing prediction.
- `example.R`: Example usage script.
